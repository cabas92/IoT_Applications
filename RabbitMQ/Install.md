  ### Erlang/OTP
  It's necessary for the operation of rabbitmq
  ```
  sudo apt-get update
  sudo apt-get install -y erlang logrotate
  sudo apt-get install
  ```
  or
  ```
  sudo apt-get update
  sudo apt-get install erlang-nox
  sudo apt-get install
  ```
  If you can, previously to the instalation upgrade the system.
